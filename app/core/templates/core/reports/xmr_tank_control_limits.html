Tank Leak Detection Control Limits

Wheeler's Statistical Process Control limits for tank leak detection during non-operation hours

How Leak Detection Works
Wheeler's XmR Charts (Individuals & Moving Range)

This system uses Wheeler's Statistical Process Control method to detect abnormal tank level changes that may indicate leaks:

  • X Chart (Individuals): Tracks hourly tank level changes during non-operation hours
  • mR Chart (Moving Range): Tracks the volatility of changes between consecutive hours
Control Limit Formulas
Upper Control Limit (UCL)= Average Change + 2.66 × Average Moving Range
Lower Control Limit (LCL)= Average Change - 2.66 × Average Moving Range

The constant 2.66 is Wheeler's scaling factor for individual measurements (d2 = 1.128 for n=2, so 3/d2 ≈ 2.66).

Non-Operation Hours

Limits are calculated using only data from times when production should be inactive:

  • Weekdays: 6:00 PM to 3:00 AM (Central Time)
  • Weekends: All day Saturday and Sunday
Leak Detection Rules
  1. Point Below LCL: If the current rate of change drops below the Lower Control Limit, it may indicate a leak
  2. 6+ Consecutive Decreases: If tank levels decrease for 6 or more consecutive hours, this signals a potential leak
